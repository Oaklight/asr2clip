# 持续录音模式

持续录音模式适用于会议、讲座或访谈等长时间录音场景。

## 固定间隔模式

按固定时间间隔转录：

```bash
# 每 60 秒转录一次
asr2clip --interval 60 -o ~/meeting.txt
```

## VAD + 间隔

将语音活动检测与最大间隔结合：

```bash
# 静音时自动转录，最长间隔 120 秒
asr2clip --vad --interval 120 -o ~/meeting.txt
```

## 行为特点

持续模式下：

- 持续录音
- 自动转录（静音时或达到间隔时）
- 转录**异步执行** — 上一段音频转录的同时继续录音，结果按顺序输出
- 按一次 **Ctrl+C** 停止 — 退出前会转录剩余音频
- 按两次 **Ctrl+C** 强制立即退出
- 转录结果带时间戳追加到输出文件

## 使用建议

- 始终使用 `-o FILE` 保存转录结果 — 剪贴板只保留最新一条
- 结合 `--vad` 实现自然语音边界分段
- 使用 `--interval` 作为兜底，确保长时间沉默不会遗漏内容
